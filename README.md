# Список решённых задач

В этом репозитории собраны все решённые задачи по практикуму на ЭВМ, преподаватель Арушанян Игорь Олегович. Все программы собираются и запускаются при помощи `make full`, просто запуск `make run`.

## Список задач
1. **Нахождении минимума функции** – программа находит точку минимума функции, указанной в файле Functions.cpp. В программе реализованы метод дробления шага для первичного приближения с точностью sqrt(epsilon) и последующего приближения методом золотого сечения с точностью epsilon.

2. **Решение интегрального уравнения Фредгольма** - программа решает уравнение Фредгольма, заданнное в файле Functions.cpp, с использованием метода трапеций(программа написана не полностью, метод должен быть реализован при помощи квадратур(вроде)).
